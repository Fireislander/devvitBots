# Privacy Policy for nhlgdtbot

Last Updated: May 25, 2026

## 1. Information We Collect
The nhlgdtbot is a Reddit bot designed to schedule and update NHL Game Day Threads. We do not collect, store, or share any personal user data, private messages, or account credentials.

## 2. Third-Party Services
This application fetches public game data from the NHL API (api-web.nhle.com) to generate game threads. No user data is sent to the NHL or any other third-party API.

## 3. Contact
If you have any questions or feedback, please contact the developer via Reddit PM.
